# Introduction

Cypress is an E2E open source used to automate scripts.

## Installation

Use the cypress io [Cypress interface](https://www.cypress.io/) to install .

```bash
npm install cypress --save-dev
```
Then lauch the app .

```bash
npx cypress open
```
